# Hibernate HQL Project

## Overview
This project demonstrates the use of Hibernate Query Language (HQL) with Java and Hibernate Framework.

## Technologies Used
- Java
- Hibernate ORM
- MySQL
- Maven

## Features
- Insert Student Records
- Fetch Student Data
- Execute HQL Queries
- Single Result Queries
- Database Connectivity using Hibernate
